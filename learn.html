Learn C++

1. Introduction to C++



C++ is a powerful, high-performance programming language used in various fields like game development, software engineering, and competitive programming. It supports both procedural and object-oriented programming, making it versatile and efficient.



2. Setting up the Environment



Before you begin coding, you need a C++ compiler. You can use:

  • IDE like Code::Blocks, Visual Studio Code, or CLion.
  • Text Editor with a command-line compiler like g++ for Linux or MinGW for Windows.


Once set up, create a new file with the extension .cpp, and you're ready to write your first C++ program!



3. Hello, World! Program



Let's start with a basic "Hello, World!" program to understand the structure of a C++ program.


            #include <iostream> // Preprocessor directive for input and output

            int main() {
                std::cout << "Hello, World!" << std::endl; // Prints message to the console
                return 0; // Indicates successful program execution
            }

Explaination:

  • #include <iostream>: This includes the input-output stream library.
  • int main(): This is the main function where the program starts executing.
  • std::cout: Used to output (print) text to the console.
  • return 0;: Returns 0 to indicate the program executed successfully.


4. Variables and Data Types



Variables store data that can be used later in the program. Common data types in C++ include:

  • int: for integers.
  • float: for decimal numbers.
  • char: for characters.
  • bool: for true/false values.
  • string: for text (requires including the #include <string> header).

Example:


            #include <iostream>

            int main() {
                int age = 20;
                float height = 5.9;
                char grade = 'A';
                bool isStudent = true;
            
                std::cout << "Age: " << age << std::endl;
                std::cout << "Height: " << height << std::endl;
                std::cout << "Grade: " << grade << std::endl;
                std::cout << "Is Student: " << isStudent << std::endl;
            
                return 0;
            }



5. Input and Output


You can use cin for taking input from the user and cout for output.


Example:

            #include <iostream>

            int main() {
                int num;
                
                std::cout < "Enter a number: ";
                std::cin >> num; // Takes input from the user and stores it in 'num'
                
                std::cout < "You entered: " < num < std::endl; // Outputs the value of 'num'
            
                return 0;
            }



6. Control Structures


C++ provides several control structures like if-else, loops, and switch to control the flow of the program.

1. if-else Statements

Used to make decisions in your program based on conditions.

Example:


            #include <iostream>

            int main() {
                int age;
                std::cout < "Enter your age: ";
                std::cin >> age;
            
                if (age >= 18) {
                    std::cout << "You are eligible to vote." << std::endl;
                } else {
                    std::cout << "You are not eligible to vote." << std::endl;
                }
            
                return 0;
            }

2. For Loops

Used to repeat a block of code a certain number of times.

Example:


            #include <iostream>

            int main() {
                for (int i = 1; i <= 5; i++) {
                    std::cout << "Count: " << i << std::endl;
                }
            
                return 0;
            }

3. While Loops

A loop that continues to execute as long as a condition is true.

Example:

            #include <iostream>

            int main() {
                int count = 1;
                
                while (count <= 5) {
                    std::cout << "Count: " << count << std::endl;
                    count++;
                }
            
                return 0;
            }



7. Functions


Functions are blocks of reusable code. They help structure your code and avoid repetition.

Example:

            #include <iostream>

            // Function to add two numbers
            int add(int a, int b) {
                return a + b;
            }
            
            int main() {
                int num1, num2;
                std::cout << "Enter two numbers: ";
                std::cin >> num1 >> num2;
            
                int result = add(num1, num2); // Function call
                std::cout << "Sum: " << result << std::endl;
            
                return 0;
            }



8. Arrays


Arrays are collections of elements of the same type, stored at contiguous memory locations.

Example:

            #include <iostream>

            int main() {
                int arr[5] = {1, 2, 3, 4, 5};
            
                for (int i = 0; i < 5; i++) {
                    std::cout << "Element at index " << i << ": " << arr[i] << std::endl;
                }
            
                return 0;
            }



9. Conclusion


This tutorial covers the basics of C++. Understanding these concepts is essential for further exploration of advanced topics such as object-oriented programming (OOP), pointers, and file handling.



Start experimenting with these examples, and soon you'll be confident in writing your own C++ programs!
